# LangChain Models

Simple examples of using **LangChain** with **Google Gemini** and **Hugging Face**.

## Setup

1.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Add API keys** to a `.env` file:
    ```env
    GOOGLE_API_KEY=your_key
    ```

## Usage

Run the scripts in each folder to see LLMs, Chat Models, and Embeddings in action.

```bash
python 1.LLM/main.py
python 2.ChatModels/main.py
python 3.EmbeddingModels/main_similarity.py
```
